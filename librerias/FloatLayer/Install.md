# 📦 Guía Rápida de Instalación (JSR)

¡**FloatLayer** se distribuye a través del registro moderno **JSR** para garantizar soporte de TypeScript nativo, tipados súper limpios sin archivos `.d.ts` pesados y compatibilidad multiplataforma!

A continuación, tienes los comandos exactos según el gestor de paquetes de tu preferencia:

---

## 🦕 Deno

### Agregar paquete
```bash
deno add jsr:@alejandro/floatlayer
```

### Importar Símbolo
```typescript
import * as floatlayer from "@alejandro/floatlayer";
```

*O importando directamente usando el especificador de JSR sin pre-instalación:*
```typescript
import * as floatlayer from "jsr:@alejandro/floatlayer";
```

---

## 📦 pnpm

### Instalar mediante JSR
```bash
pnpm i jsr:@alejandro/floatlayer
```

*O (si usas pnpm 10.8 o versiones más antiguas):*
```bash
pnpm dlx jsr add @alejandro/floatlayer
```

### Importar Símbolo
```typescript
import * as floatlayer from "@alejandro/floatlayer";
```
