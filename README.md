# 🌌 Dev Library Hub (Portfolio de Ingeniería de Software)

¡Bienvenido a **Dev Library Hub**! Este es un portal interactivo y visualmente pulido diseñado como un catálogo/sandbox inteligente para desarrolladores que valoran el código limpio, el rendimiento del hardware, la resiliencia en red y el diseño soberano.

El proyecto está diseñado bajo la propuesta estética **"Elegant Dark"**, utilizando contrastes altos, tipografía limpia y micro-interacciones robustas impulsadas por `Framer Motion`.

---

## 🚀 Propósito del Proyecto

Este hub no es un proyecto de demostración pasivo; es una plataforma de experimentación en caliente fabricada con **Next.js 15+ (App Router)** y **Tailwind CSS v4** donde los desarrolladores pueden interactuar con simuladores reales de librerías modernas de código abierto:

1.  **State Tapestry**: Manejador reactivo de estados atómicos con Time-Travel (viaje en el tiempo).
2.  **Fetch Magic WASM**: Simulador de resiliencia de red con reintentos y retrocesos exponenciales adaptativos (*Exponential Backoff*).
3.  **Glassy View**: Diseños con esmerilado de vidrio CSS filtrado de alta performance.
4.  **Omni Hooks (Zen Animate)**: Simulador de resortes físicos (*Spring Physics*) manipulando masas y amortiguación en tiempo real.
5.  **Kotlite DB (Nueva incorporación)**: Base de datos local inmutable inspirada en el DSL de **Kotlin** corriendo de forma segura en Next.js (SSR Safe).

---

## 🛠️ Tecnologías Utilizadas

*   **Framework:** Next.js 15 (App Router)
*   **Lenguaje:** TypeScript (Tipado estricto)
*   **Estilos:** Tailwind CSS v4 para diseño fluido y ultra responsivo.
*   **Animaciones:** Framer Motion (`motion/react`) para fluidez cinemática.
*   **Iconos:** Lucide React

---

## 📦 Inicialización y Ejecución Local

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```

2.  **Levantar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

3.  **Compilar para producción:**
    ```bash
    npm run build
    ```
